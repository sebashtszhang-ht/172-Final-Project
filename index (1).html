EEC 172 Final Project

Security Node

A standalone smart security system built on the CC3200 LaunchPad. The device combines local sensing, on-device interaction, and AWS IoT alerts without requiring any new hardware.

Project Overview Deliverables
Inputs IR + Motion
Output OLED + AWS
Core Idea Authenticate locally, detect movement, then escalate to cloud alerts.

Overview

What the project does

Security Node starts in DISARMED. A user enters a 4-digit PIN with the IR remote and confirms with MUTE to arm the system. While armed, the onboard accelerometer monitors movement.

If motion exceeds the threshold, the board enters a countdown state and gives the user a short window to disarm. If the countdown expires, the device posts an alert to AWS IoT Thing Shadow and can trigger an SNS notification.

Implemented features

  • 4-digit IR PIN entry with delete and confirm
  • OLED interface for status, PIN input, and alert state
  • Accelerometer-based motion trigger on the CC3200 board
  • 10-second countdown before alarm escalation
  • AWS IoT Thing Shadow update after an alarm event
  • Standalone boot flow from flashed firmware

Architecture

How the system is organized

Local Interaction

The IR receiver captures remote button presses. Users enter a PIN, delete one digit with LAST, and confirm with MUTE.

Embedded State Machine

The firmware manages transitions between disarmed, armed, triggered, and alert states, keeping the user interface and cloud updates consistent.

Cloud Escalation

When the countdown expires, the board sends a secure HTTPS request to AWS IoT Thing Shadow. An AWS rule can forward the event through SNS.

Demo flow, files, and publishing links

Demo sequence

  1. Boot the board and show the DISARMED screen.
  2. Enter a valid 4-digit PIN with the IR remote and arm the device.
  3. Move the board to trigger the accelerometer threshold.
  4. Show the 10-second countdown on the OLED.
  5. Let the countdown expire and verify the AWS IoT update.
  6. Disarm the system with the correct PIN and return to idle.
